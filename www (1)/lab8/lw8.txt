=====================================================================
Задания.
---------------------------------------------------------------------
1.  Разработайте механизм авторизации и регистрации на сайте. 
#40

2.  Усовершенствуйте PHP скрипты Survey Saver и Survey Info таким образом,
#20 чтобы информация сохранялась в базе данных MySQL.

3.  Доработайте PHP скрипт для загрузки файлов на сервер. Файлы должны 
#25 сохраняться в директории /uploads/. 
    Орагнизуйте сохранение данных о загруженных файлах в базе данных.
    Имя файла в файловой системе должно быть <file_id>.<extension>
    Реализуйте связь данных о загруженных файлах с данными пользователя,
    который их добавил.

-- Дополнительные задания --

4.  Добавьте страницу со списком пользоваьелей Survey List. На странице 
#10 необходимо отображать данные пользователей Email, First Name, Last Name.

5.  Организуйте связь данных списка пользователей Survey List со страницей 
#10 Get Survey Info. Для этого сделайте Email пользователя ссылкой, которая 
    ведет на страницу Get User Info.
    В ссылке необходимо передавать id пользователя.
    На странице Get User Info необходимо добавить проверку переданных 
    параметров GET. Если в параметрах есть id пользователя, то необходимо
    сразу показывать его данные без ввода email'a.

6.  Модифицируйте страницу Survey List таким образом, чтобы в списке отображалось
#20 не более 20 элементов (должно задаваться в конфигурационном файле в виде define).
    Организуйте постраничную навигацию.

7.  Разработайте PHP скрипт формирующий список загруженных на сервер файлов.
#20 Добавьте форму для фильтрации данных.
    На форме должны быть поля User Email и File Extension и кнопка Filter 
    для отправки формы.
    Если задано лишь одно поле, то необходимо фильтровать только по нему.
